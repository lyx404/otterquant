# Launch Guide Design Notes

## Reference (Alpaca Launch Guide)
- Title: "Launch Guide" in bold serif/sans at top
- Horizontal stepper with 4 steps: Welcome → Sandbox → Limited Live Requirements → Live Requirements
- Current step has green circle, future steps have gray circles
- Steps connected by horizontal line (green for completed, gray for pending)
- Left sidebar has collapsible chevrons (accordion sections)
- Form fields: text input with dark bg, dropdown selects, radio buttons
- "Next" button at bottom left in olive/gold color
- Overall dark background, minimal borders

## Our Steps (adapted for AlphaForge)
1. Welcome — Team name, trading experience, preferred markets
2. Install Skill — Guide to install AI Agent skill (with code snippet / instructions)
3. Verify Installation — Check if skill connected successfully, show status
4. First Run — Guide to create first Alpha factor, submit to backtest
