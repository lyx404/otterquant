## 方案一：Terminal Noir（终端黑客美学）

**设计运动**：Cyberpunk Terminal / Hacker Aesthetic — 灵感来自彭博终端、黑客帝国的数据流，以及专业量化交易员的工作环境。

**核心原则**：
1. 信息密度优先 — 每个像素都承载数据，不浪费空间在装饰上
2. 单色层次 — 通过灰度和荧光色的对比建立视觉层级
3. 等宽字体即权威 — 数字和代码使用等宽字体传达专业感
4. 实时感 — 所有数据看起来都在"活着"，有微妙的脉动和更新动画

**色彩哲学**：以纯黑 (#0A0A0F) 为底，用荧光绿 (#00FF88) 作为主要强调色（代表盈利/正向），荧光红 (#FF3366) 代表风险/亏损，冷蓝 (#4ECDC4) 用于中性信息。这种配色模拟了专业交易终端的视觉语言，让用户感到自己在操控一个高端系统。

**布局范式**：固定侧边栏导航 + 可拖拽的面板系统。主内容区采用多面板网格，用户可以自由调整每个面板的大小。类似 Bloomberg Terminal 的信息密集布局。

**签名元素**：
- 所有数字使用 JetBrains Mono 等宽字体，带有微妙的发光效果
- 图表使用扫描线动画效果，数据点像雷达一样被"扫描"出来
- 卡片边框使用 1px 的荧光色描边，hover 时发光加强

**交互哲学**：快速、精准、无废话。鼠标悬停即显示详细数据tooltip，点击即展开，双击即全屏。所有操作都有键盘快捷键。

**动画**：数据加载时使用矩阵式的数字滚动效果；页面切换使用快速的滑入；图表数据使用逐点绘制动画。

**字体系统**：标题使用 Space Grotesk（几何无衬线，现代感强），数据使用 JetBrains Mono（等宽，专业），正文使用 IBM Plex Sans（清晰可读）。

## 方案二：Quantitative Elegance（量化优雅）

**设计运动**：Swiss Design / Data Visualization Art — 灵感来自 Edward Tufte 的数据可视化理论、瑞士国际主义设计，以及高端金融机构的品牌设计。

**核心原则**：
1. 数据即美 — 图表本身就是视觉焦点，不需要额外装饰
2. 留白即层次 — 通过精心计算的间距建立信息层级
3. 克制的色彩 — 颜色只在需要传达含义时才出现
4. 排版即结构 — 字体大小和粗细的变化承担了大部分视觉组织工作

**色彩哲学**：以温暖的米白 (#FAFAF8) 为底色，深墨色 (#1A1A2E) 为主要文字色。强调色使用深蓝 (#16213E) 和琥珀色 (#E8B931)。图表中训练期使用靛蓝 (#4361EE)，测试期使用珊瑚橙 (#F7A072)。整体传达"安静的自信"和"学术的严谨"。

**布局范式**：左侧固定导航栏（窄，图标+文字）+ 右侧内容区采用报纸式的多栏布局。重要数据使用大号字体在页面顶部形成"头条"效果，详细数据在下方以精心排列的卡片和表格呈现。

**签名元素**：
- 关键指标数字使用超大号字体（48-72px），配合极细的标签文字，形成强烈的大小对比
- 图表区域使用极简的坐标轴，数据线条流畅优雅，没有多余的网格线
- 状态指示使用小圆点而非色块，克制而精确

**交互哲学**：优雅而从容。悬停时信息缓缓浮现，点击时内容平滑展开。没有突兀的弹窗，所有详情都在页面内优雅地展开和收起。

**动画**：页面加载时数字从0滚动到目标值（缓动函数使用 ease-out-expo）；图表线条使用从左到右的描边动画；卡片使用微妙的上浮入场效果。

**字体系统**：标题使用 Playfair Display（衬线体，传达金融的传统与权威），数据数字使用 DM Mono（等宽但更优雅），正文使用 Source Sans 3（清晰中性）。

## 方案三：Neon Dashboard（霓虹仪表盘）

**设计运动**：Glassmorphism + Neon Accent — 灵感来自现代加密货币交易所（如 dYdX、GMX）的 UI 风格，融合毛玻璃效果和霓虹色彩。

**核心原则**：
1. 深色底+玻璃层 — 深色背景上叠加半透明毛玻璃面板，创造深度感
2. 霓虹引导 — 关键操作和重要数据使用霓虹色高亮，引导视线
3. 圆润现代 — 大圆角、柔和阴影，传达友好但专业的感觉
4. 渐变即品牌 — 品牌色使用蓝紫渐变，与加密货币社区的视觉语言共鸣

**色彩哲学**：深蓝黑底 (#0F1729)，面板使用半透明白 (rgba(255,255,255,0.05))。主品牌色使用蓝到紫的渐变 (#3B82F6 → #8B5CF6)。盈利绿 (#10B981)，亏损红 (#EF4444)。这种配色在加密货币用户群体中有天然的亲和力。

**布局范式**：顶部导航栏 + 侧边栏组合。内容区使用卡片式布局，每个卡片都是一个毛玻璃面板。卡片之间有充足的间距，整体感觉像是在深色空间中漂浮的信息面板。

**签名元素**：
- 毛玻璃卡片带有微妙的边框光晕效果（border: 1px solid rgba(255,255,255,0.1)）
- 重要按钮使用渐变背景 + 外发光效果
- 图表背景使用渐变填充区域，营造"发光"的数据感

**交互哲学**：流畅而有反馈。按钮点击有弹性缩放效果，卡片悬停时边框发光加强，切换标签页时内容有平滑的淡入淡出。

**动画**：卡片入场使用交错的上浮动画（stagger）；数字变化使用弹性动画；图表使用渐进式绘制 + 区域渐变填充动画。

**字体系统**：标题使用 Sora（几何无衬线，现代科技感），数据使用 Space Mono（等宽，清晰），正文使用 Plus Jakarta Sans（温暖可读）。
